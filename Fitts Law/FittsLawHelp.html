Fitts' Law Help

What?

This application helps us to obtain Fittsâ€™ parameters by using random coordinate changes of a circle inside a canvas.

How?

Set the initial size for the circle and press the "Start" button. Then try to click on the circle as fast as possible. Each 5 clicks cause the circle to change size. After a desired number of tries press the "Compute" button in order to calculate the MT (Fitts' time)
